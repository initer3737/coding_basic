pencarian vim 
tekan tombol s atau / lalu di enter
n untuk me hightlight kata yang di cari next [maju] 
sebelumnya tekan N untuk pencarian [mundur] 
pencarian dilakukan/dimulai di posisi kursor yang terakhir aktif di line tertentu
gg [pindah kursor ke baris pertama] n untuk mencari next search result
G untuk ke baris terakhir 

untuk mempermudah tanda tekan tombol s atau / maka kerahkan kursor ke tulisan yang akan dicari lalu tekan tombol seperti * atau # 
*digunakkan pencarian kata ke depan # ke belakang

:Explore atau :E: Membuka explorer di jendela saat ini.
:Vexplore atau :Vex: Membuka explorer di vertikal split (sisi kiri).
:Hexplore atau :Hex: Membuka explorer di horizontal split.
Perintah di dalam Netrw:
Enter: Masuk ke folder atau membuka file.
-: Kembali ke direktori sebelumnya (naik satu folder).
d: Membuat direktori baru.
%: Membuat file baru.
R: Mengubah nama file/folder. 
2. Menggunakan Perintah Terminal
nvim .: Membuka Neovim langsung pada direktori saat ini.
:e .: Membuka file explorer di buffer aktif saat sedang mengedit. 
3. Alternatif (Plugin NvimTree)
Jika Anda menggunakan Neovim dengan konfigurasi seperti NvChad, biasanya menggunakan plugin NvimTree.
Ctrl + n: Membuka atau menutup NvimTree dalam mode Normal. 
4. Navigasi Dasar
Gunakan tombol panah atau h, j, k, l untuk berpindah antar file/folder di netrw.
Ctrl-o: Berguna untuk kembali ke lokasi sebelumnya. 
Dengan menguasai :Ex atau :Vex, Anda bisa dengan mudah menjelajahi struktur folder tanpa perlu meninggalkan Neovim.
